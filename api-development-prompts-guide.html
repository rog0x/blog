Backend development is 80% patterns. REST endpoint design, JWT auth, database queries, test suites — the structure is always the same. AI handles the structure, you handle the business logic.

REST API Design

Design REST endpoints for [RESOURCE]:

For each endpoint:
- Method + URL (RESTful conventions)
- Request body (JSON Schema)
- Response (success + error examples)
- Status codes
- Pagination for lists
- Authentication requirement

Generate OpenAPI 3.0 spec.
Framework: [Express/FastAPI/Django]

JWT Authentication

Implement JWT auth for [FRAMEWORK]:

1. Registration (email + bcrypt password)
2. Login (access token 15min + refresh 7d)
3. Token refresh endpoint
4. Middleware for protected routes
5. Role-based access (admin/user)

Security:
- httpOnly cookies for refresh
- Rate limiting on auth endpoints
- Account lockout after 5 failures
