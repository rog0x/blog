30 MCP Servers, 145+ Tools, One Developer

March 20, 2026 · 3 min read · MCPMilestoneOpen Source

I hit 30 MCP servers today. All TypeScript, all MIT licensed, all zero API keys required. Here's the complete collection organized by domain.

Web & Content (4)

  • web-tools — extract, search, monitor, HTML→markdown
  • seo-tools — meta, headings, links, keywords, speed, sitemaps
  • markdown-tools — TOC, lint, format, stats, tables
  • image-tools — metadata, favicon, OG image, responsive

Development (7)

  • github-tools — repos, PRs, issues, releases, contributors
  • api-tools — HTTP requests, JWT, headers, health checks
  • npm-tools — search, info, audit, bundle size
  • git-tools — log, diff, blame, stats, branches
  • testing-tools — test gen, mock data, coverage, assertions
  • docs-tools — JSDoc, README, changelog, API docs

  • string-tools — analyzer, transformer, diff, templates
  • database-tools — SQL format, explain, schema, migrations
  • config-tools — JSON, YAML, TOML, INI, diff

DevOps & Infrastructure (5)

  • docker-tools — containers, images, Dockerfile, compose
  • env-tools — .env parser, ports, system info, deps
  • network-tools — DNS, IP, SSL, WHOIS, ping
  • log-tools — parse, filter, analyze, format
  • perf-tools — benchmark, memory, Big O, load test

Code Quality (2)

  • lint-tools — style, naming, complexity, dead code
  • a11y-tools — alt text, headings, contrast, ARIA, forms

Utilities (5)

  • color-tools — convert, palette, contrast, gradients
  • crypto-tools — hash, encode/decode, UUID, passwords
  • time-tools — timezone, dates, cron, timestamps
  • math-tools — calculator, units, statistics
  • file-tools — dir tree, stats, duplicates, LOC

Specialized (2)

  • i18n-tools — language detect, translations, plurals
  • cache-tools — cache headers, CDN, simulation

All servers follow the same pattern: 5 focused tools per server, TypeScript + @modelcontextprotocol/sdk, zero external API keys.
