How to Install Any Server

Clone and run:

git clone https://github.com/rog0x/mcp-web-tools
cd mcp-web-tools && npm install && npm run build
claude mcp add web-tools node ./dist/index.js

Why 20 Servers?

Each server is focused on one domain. This means:

  • Install only what you need
  • Each server is small and fast
  • Easy to maintain and update
  • No bloated monolith
