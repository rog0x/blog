Build Your First MCP Server in 30 Minutes

March 20, 2026 · 10 min read · MCPTutorialTypeScript

MCP servers let you give Claude superpowers. In this tutorial, you'll build one from scratch — a word counter that analyzes text — and connect it to Claude Code.

Prerequisites

  • Node.js 18+ installed
  • TypeScript (npm install -g typescript)
  • Claude Code installed

Step 1: Set Up the Project

mkdir my-mcp-server
cd my-mcp-server
npm init -y
npm install @modelcontextprotocol/sdk
npm install -D typescript @types/node
npx tsc --init

Update tsconfig.json:

{
  "compilerOptions": {
    "target": "ES2022",
    "module": "Node16",
    "moduleResolution": "Node16",
    "outDir": "./dist",
    "rootDir": "./src",
    "strict": true,
    "esModuleInterop": true
  },
  "include": ["src/**/*"]
}

Step 2: Create the Server

Create src/index.ts:

#!/usr/bin/env node
import { Server } from "@modelcontextprotocol/sdk/server/index.js";
import { StdioServerTransport }
  from "@modelcontextprotocol/sdk/server/stdio.js";
import {
  CallToolRequestSchema,
  ListToolsRequestSchema,
} from "@modelcontextprotocol/sdk/types.js";

// Create the server
const server = new Server(
  { name: "word-counter", version: "1.0.0" },
  { capabilities: { tools: {} } }
);

// List available tools
server.setRequestHandler(
  ListToolsRequestSchema,
  async () => ({
    tools: [{
      name: "count_words",
      description: "Count words, sentences, and paragraphs",
      inputSchema: {
        type: "object",
        properties: {
          text: {
            type: "string",
            description: "The text to analyze"
          }
        },
        required: ["text"]
      }
    }]
  })
);

// Handle tool calls
server.setRequestHandler(
  CallToolRequestSchema,
  async (request) => {
    const { name, arguments: args } = request.params;

    if (name === "count_words") {
      const text = args?.text as string;
      const words = text.split(/\s+/).filter(Boolean).length;
      const sentences = text.split(/[.!?]+/).filter(Boolean).length;
      const paragraphs = text.split(/\n\n+/).filter(Boolean).length;
      const readingTime = Math.ceil(words / 200);

      return {
        content: [{
          type: "text",
          text: JSON.stringify({
            words,
            sentences,
            paragraphs,
            characters: text.length,
            readingTimeMinutes: readingTime
          }, null, 2)
        }]
      };
    }

    throw new Error(`Unknown tool: ${name}`);
  }
);

// Start
async function main() {
  const transport = new StdioServerTransport();
  await server.connect(transport);
}

main().catch(console.error);

Step 3: Build and Test

# Build
npx tsc

# Add to Claude Code
claude mcp add word-counter node ./dist/index.js

Now in Claude Code, ask: "Use word-counter to analyze this text: [paste any text]"

Claude will call your tool and show the word count, sentence count, and reading time.

Step 4: Add More Tools

Add a second tool to the same server. In the ListToolsRequestSchema handler, add another tool to the array. In the CallToolRequestSchema handler, add another case.

That's the pattern: register tools, handle calls, return results. Every MCP server follows this structure.
