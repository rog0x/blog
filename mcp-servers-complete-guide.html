Install All

Clone any server and point Claude at it:

# Clone
git clone https://github.com/rog0x/mcp-web-tools
cd mcp-web-tools && npm install && npm run build

# Add to Claude Code
claude mcp add web-tools node ./dist/index.js

Or when published on npm (coming soon):

claude mcp add web-tools npx @rog0x/mcp-web-tools
